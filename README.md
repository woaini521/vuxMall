# vuxMall

基于vux的销售印刷物料以及印刷服务微信公众号商城。前端基于vue、vux、vuex、vue-router,后端基于Java。(url需要在微信内置浏览器打开)

## Install

``
$ npm run init //编译静态文件
``

``
$ npm run dev //启动本地服务
``
